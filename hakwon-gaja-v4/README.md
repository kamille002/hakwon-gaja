# 학원가자 V4 - 온보딩 + 다자녀 지원! 👨‍👩‍👧‍👦

## 🎉 V4 주요 기능

### 1. 완전한 온보딩 시스템 ✨
최초 실행 시 4단계 가이드:
- 1단계: 환영 화면
- 2단계: 자녀 이름 입력
- 3단계: 추가 자녀 등록 (선택)
- 4단계: 완료!

### 2. 다자녀 지원 시스템 👦👦
- 자녀별 독립 데이터 (학원, 포인트, 출석률)
- 헤더 드롭다운으로 간편 전환
- 언제든지 자녀 추가 가능

### 3. 기존 기능 완벽 유지 ✅
- 날씨 알림 (비/미세먼지)
- 데이터 저장 (LocalStorage)
- PWA (홈 화면 추가)
- 학원/보상 관리
- 포인트 시스템

---

## 📱 사용 시나리오

### 첫 사용 (엄마)
1. 앱 다운로드
2. "시작하기" 클릭
3. 민준 이름 입력 → 다음
4. "자녀 추가" → 민수 입력
5. "시작하기" 클릭
6. 민준 선택 → 학원 추가
7. 민수로 전환 → 학원 추가

### 일상 사용
- 헤더에서 자녀 클릭 → 드롭다운
- 원하는 자녀 선택
- 해당 자녀 데이터만 표시
