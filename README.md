# 학원가자 MVP

**핵심 기능만 담은 간단한 버전!**

---

## ✅ MVP 기능

### **포함된 기능:**
- ✅ 자녀 등록 (여러 명 가능)
- ✅ 학원 추가 (이름, 시간, 요일)
- ✅ 오늘의 학원 자동 표시
- ✅ 출발 시간 알람 (브라우저 알림)
- ✅ 간단한 포인트 시스템 (+10P)
- ✅ 보상 추가/받기
- ✅ 로컬 저장 (데이터 보존)

### **제외된 기능 (나중에 추가):**
- ❌ Firebase (복잡)
- ❌ 부모-자녀 폰 분리
- ❌ GPS 추적
- ❌ 지도
- ❌ 날씨 API
- ❌ 녹음 기능

## 🚀 배포 방법 (초간단!)

### **1. Netlify Drop (가장 쉬움!)**

```
1. https://app.netlify.com/drop 접속
2. 이 폴더 전체를 드래그 앤 드롭!
3. 끝! 🎉
```

**주의:** 폴더 자체를 드래그하지 말고, **폴더 안의 파일들**을 드래그하세요!

```
❌ hakwon-gaja-mvp 폴더 드래그
✅ index.html, manifest.json 등 파일들 드래그
```

### **2. Vercel (추천!)**

```bash
# 설치
npm install -g vercel

# 배포
vercel

# 완료!
```

### **3. GitHub Pages**

```
1. GitHub에 저장소 생성
2. 파일 업로드
3. Settings → Pages → Deploy from main branch
4. 완료!
```

## 🎯 사용 방법

### **1단계: 자녀 추가**
```
앱 실행 → 자녀 이름 입력 → 추가
```

### **2단계: 학원 추가**
```
+ 학원 추가 클릭
→ 학원 이름 입력
→ 요일 선택
→ 출발 시간 입력 (알람 시간)
→ 수업 시간 입력
→ 추가!
```

### **3단계: 알람 확인**
```
출발 시간이 되면 → 브라우저 알림!
→ "출발했어요!" 버튼 클릭
→ +10P 획득!
```

### **4단계: 보상**
```
+ 보상 추가
→ 보상 이름 (예: 편의점 간식)
→ 필요 포인트 (예: 100P)
→ 포인트 모으면 받기!
```

## 🎨 특징

### **간단함**
- ✅ 설치 불필요 (웹앱)

## 📞 문제 해결

### **알람이 안 울려요**
```
1. 브라우저 알림 권한 확인
2. 시간 설정 다시 확인
3. 요일 제대로 선택했는지 확인
```

### **데이터가 사라졌어요**
```
1. 브라우저 캐시 삭제하면 데이터 삭제됨
2. 같은 브라우저 사용해야 함
3. 백업 기능 추가 예정
```

### **폰에서 안 보여요**
```
1. 모바일 브라우저에서 접속
2. 화면이 작으면 잘림
3. PWA 설치 권장
```
